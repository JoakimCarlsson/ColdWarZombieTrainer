# ColdWarZombieTrainer
Cold War Zombie Trainer

### Working Features:
* God Mode
* Speed Hack
* Infinite Ammo
* Infinite Money
* Insta kill ( 1 hp zombies )
* Teleport Zomies too crosshair.
* Teleport Zombies too set location.
* Always Crit
* Rapid Fire
* Thermal Vison 
* XP Multiplier
* Gun XP Multiplier 
* TimeScale Multiplier 
* Weapon Switch.
